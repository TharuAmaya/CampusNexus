CampusNexus is a comprehensive, centralized management system designed to streamline university operations. It provides a seamless digital experience for students, staff, and technicians to handle daily campus activities, from reserving facilities to reporting maintenance issues.

## ✨ Core Modules & Features

* **🔐 Authentication & User Management:** Secure Role-Based Access Control (RBAC) using Spring Security and JWT. Supports profiles for Admins, Students, and Technicians.
* **🏢 Facilities Management:** Admins can manage campus facilities, view availability, and maintain records of halls, labs, and equipment.
* **📅 Booking Management:** Students and staff can easily request, schedule, and manage facility reservations with real-time conflict checking.
* **🛠️ Incident Ticketing:** A streamlined helpdesk where users can report issues (e.g., broken AC, network failure), track ticket status, and technicians can update task progress.
* **🔔 Notifications:** Real-time updates and alerts for booking confirmations, ticket resolutions, and system announcements.

## 🛠️ Technology Stack

**Frontend:**
* React (Vite)
* Tailwind CSS (for styling)
* React Router (for navigation)

**Backend:**
* Java 17+
* Spring Boot 3.x
* Spring Security (JWT / OAuth2)
* Spring Data JPA / Hibernate

**Database:**
* MySQL (Aiven Cloud / Local)
